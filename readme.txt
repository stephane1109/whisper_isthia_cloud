# Transcription d'audio avec whisper - OpenAI

## Utilisation

- Choisissez la source de l'audio (URL YouTube ou Fichier MP3).
- Configurez les options (taille du modèle, langue, mode débug).
- Cliquez sur **Lancer la transcription**.  
  La transcription est affichée dans l'application et enregistrée automatiquement dans un fichier texte dont le chemin est indiqué.
